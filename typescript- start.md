рЃарЃљ рЃЌрЃЦрЃЏрЃљ рЃБрЃюрЃЊрЃљ! рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃљрЃарЃўрЃА рЃАрЃарЃБрЃџрЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљ **рЃюрЃљрЃЉрЃўрЃ»-рЃюрЃљрЃЉрЃўрЃ»**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃАрЃгрЃљрЃЋрЃџрЃўрЃА рЃарЃЮрЃњрЃЮрЃа:

1. рЃЊрЃљрЃЋрЃљрЃДрЃћрЃюрЃЮрЃЌ TypeScript
2. рЃЊрЃљрЃЋрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃЮрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃў

4. рЃЕрЃљрЃЋрЃарЃЌрЃЮрЃЌ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЎрЃЮрЃЏрЃърЃўрЃџрЃљрЃфрЃўрЃљ (watch)

## РюЁ 0. рЃгрЃўрЃюрЃљрЃърЃўрЃарЃЮрЃЉрЃљ

рЃЊрЃљрЃарЃгрЃЏрЃБрЃюрЃЊрЃўрЃЌ рЃарЃЮрЃЏ рЃњрЃљрЃЦрЃЋрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў:

* **Node.js** ([https://nodejs.org](https://nodejs.org))
* **рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃў** (CMD, PowerShell, Git Bash, VSCode Terminal)

---

## РюЁ 1. рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃљ TypeScript-рЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ

```bash
mkdir ts-project
cd ts-project
npm init -y                           # рЃЦрЃЏрЃюрЃўрЃА package.json-рЃА
npm install typescript --save-dev    # рЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћрЃЉрЃА TypeScript-рЃА
npx tsc --init                        # рЃЦрЃЏрЃюрЃўрЃА tsconfig.json-рЃА
```

---

## РюЁ 2. рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃцрЃЮрЃџрЃЊрЃћрЃарЃў рЃЊрЃљ TypeScript рЃцрЃљрЃўрЃџрЃў:

```bash

```

рЃљрЃю рЃ«рЃћрЃџрЃўрЃЌ рЃерЃћрЃЦрЃЏрЃћрЃюрЃў `src/index.ts` рЃцрЃљрЃўрЃџрЃў рЃљрЃЏ рЃерЃўрЃюрЃљрЃљрЃарЃАрЃўрЃЌ:

```ts
const msg: string = "Hello TypeScript";
console.log(msg);
```

## РюЁ 3. tsconfig.json-рЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃњрЃљрЃ«рЃАрЃћрЃюрЃў `tsconfig.json` рЃЊрЃљ рЃЊрЃљрЃљрЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћ рЃљрЃАрЃћ:

```jsonc

    "target": "ES6",
    "module": "ES6",
    "rootDir": "./src",
    "outDir": "./dist",
    "strict": true,
    "esModuleInterop": true

}
```

## РюЁ 4. рЃЎрЃЮрЃЏрЃърЃўрЃџрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

**рЃћрЃарЃЌрЃ»рЃћрЃарЃљрЃЊрЃў рЃЎрЃЮрЃЏрЃърЃўрЃџрЃљрЃфрЃўрЃљ:**

**рЃњрЃљрЃЏрЃЮрЃўрЃАрЃљрЃ«рЃћрЃЉрЃљ рЃцрЃљрЃўрЃџрЃў:** `dist/index.js`

## РюЁ 5. HTML рЃцрЃљрЃўрЃџрЃў (рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў рЃњрЃљрЃерЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА)

рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃцрЃћрЃАрЃЋрЃерЃў рЃерЃћрЃЦрЃЏрЃћрЃюрЃў `index.html`:

  <body>

## РюЁ 6. Watch рЃарЃћрЃЪрЃўрЃЏрЃў (рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЎрЃЮрЃЏрЃърЃўрЃџрЃљрЃфрЃўрЃљ)

## РюЁ 8. (рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃАрЃљрЃЏрЃћрЃЉрЃа) `package.json` рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃў

## ­ЪјЅ рЃЏрЃќрЃљрЃЊ рЃ«рЃљрЃа!
